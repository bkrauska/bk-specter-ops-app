# Game Rules Reference

## Overview
Specter Ops is a hidden-movement game for 2-5 players. One player is the Agent, others are Hunters.

## Win Conditions

### Agent Wins
- Complete required number of objectives (usually 3 or 4)
- Survive until round limit (usually 40)

### Hunters Win
- Locate and eliminate the agent
- Agent elimination occurs when:
  - Hunter moves onto agent's space
  - Hunter shoots agent with line of sight

## Turn Structure

### Agent Turn
1. **Movement Phase**: Move up to character speed (typically 4 spaces)
   - Can move through walls with special ability
   - Must mark position secretly
2. **Objective Phase**: Check if on objective location
3. **Equipment Phase**: Play one equipment card (optional)

### Hunter Turn (All hunters act simultaneously)
1. **Movement Phase**: Move up to character speed (typically 3 spaces)
   - Cannot move through walls (unless ability)
   - Position is public
2. **Action Phase**: Choose one:
   - **Search**: Place marker on current location
   - **Shoot**: Target visible location within range
   - **Equipment**: Use equipment card
3. **Line of Sight Check**: Check if agent is visible

## Movement Rules

### Distance Calculation
- Orthogonal movement: 1 space = 1 movement point
- Diagonal movement: 1 space = 1 movement point
- Total movement cannot exceed character speed

### Restricted Movement
- Cannot move through walls (except with ability)
- Cannot move through other players (hunters only)
- Cannot end turn on another player's space (hunters only)

## Line of Sight

### Basic LOS Rules
- Straight line from hunter to target
- Blocked by walls
- Blocked by corners
- Maximum range varies by character (usually unlimited for vision)

### Shooting Rules
- Requires line of sight
- Range limit applies (character-specific)
- Agent must be revealed or suspected

## Equipment Cards

### Agent Equipment
- Limited use per game
- Can enhance movement, hide, or disrupt hunters
- Examples: Disguise, Flash Grenade, Smoke Bomb

### Hunter Equipment
- Limited use per game
- Can reveal agent or restrict movement
- Examples: Motion Sensor, Radar, Lockdown

## Character Abilities

### Agent Characters
- **The Shadow**: Can move through walls (1 per turn)
- **The Driver**: Enhanced movement speed (5 spaces)
- **The Beast**: Can push hunters when revealed
- **Ember**: Can teleport short distances

### Hunter Characters
- **The Hammer**: Increased shooting range
- **The Spotter**: Enhanced line of sight
- **The Sentry**: Can block areas
- **The Hacker**: Can track agent movement

## Mission Objectives

### Objective Types
1. **Location**: Reach specific board location
2. **Item**: Collect item from location
3. **Survive**: Survive until specific round

### Completion
- Agent must end movement on objective
- Some objectives require staying for multiple rounds
- Completed objectives are marked

## Round Limit
- Default: 40 rounds
- Can be adjusted in game settings
- Agent wins if survived at round limit

## Revelations

### Agent Must Reveal When:
- Hunter moves onto agent's exact location
- Hunter shoots agent's location with LOS
- Certain equipment cards are played

### Partial Reveals
- Motion sensor: Reveals if agent in area
- Radar: Reveals direction/distance
- Does not give exact location
